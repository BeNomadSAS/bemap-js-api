Style picker v2

The map is created with no style option, so it paints the tiny font-free fallback instantly and then loads the live default style from the server after login (/api/default-style → /styles/<name>). Pick any allow-listed style and map.setStyle(name) fetches /styles/<name> and swaps it in (markers/overlays survive). MapLibre only — vector styles.
